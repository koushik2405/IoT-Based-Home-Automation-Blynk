# IoT Based Home Automation using Arduino & Blynk

## Internship Project – Emertxe Information Technologies

## Description
This project demonstrates an IoT based home automation system simulated using PicsimLab.
The system allows control of lights, temperature, and water tank level using the Blynk IoT mobile application.

## Features
- Automatic garden light control using LDR
- Temperature monitoring using LM35
- Heater and cooler control via Blynk
- Water tank inlet and outlet valve control
- Threshold based alerts and notifications

## Tools & Technologies
- Arduino UNO
- PicsimLab Simulator
- Blynk IoT Application
- LM35 Temperature Sensor
- LDR
- Serial Tank Simulation
- CLCD
